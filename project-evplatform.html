React + TypeScript 기반으로 진행되었으며, 컴포넌트 단위부터 전체 페이지에 이르기까지 퍼블리싱을 전담하여 참여했습니다.
프로젝트 종료 이후에도 소규모 유지보수 및 고도화가 이어질 수 있다는 기획팀의 의견을 반영하여, 스타일링에는 styled-components를 도입하였습니다.
이를 통해 컴포넌트 단위의 유지보수성과 재사용성을 높이고, 향후 확장에 유리한 구조를 구축할 수 있었습니다.

외부 기획·디자인·개발팀과의 원활한 협업을 위해 컴포넌트 가이드 페이지를 구축하였고, 이를 통해 컴포넌트 확인 및 활용의 효율성과 일관성을 높일 수 있었습니다.
프로젝트의 핵심이었던 데이터 그리드 영역은 AG Grid를 기반으로 커스터마이징하였으며, hover 효과, 아이콘, 스위치, 셀렉트 박스 등을 사용자 요구에 맞게 구현하였습니다.
