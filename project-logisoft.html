React + TypeScript (Next.js) 기반 프로젝트로, Tailwind를 활용해 컴포넌트 단위부터 전체 페이지까지 퍼블리싱을 전담했습니다.

이 프로젝트에서는 처음으로 Tailwind를 도입하게 되어, 본격적인 개발에 앞서 사전 학습과 컴포넌트 테스트를 진행하였습니다. Tailwind는 직관적인 클래스 구조 덕분에 빠르게 이해하고 적용할 수 있었으나, 클래스명이 길어짐에 따라 가독성이 떨어지는 단점도 있었습니다. 이를 보완하기 위해 클래스 순서를 정해 규칙적으로 정리하는 작업 방식으로 통일감을 유지하며 효율적으로 퍼블리싱을 진행하였습니다.
