릴로 랜딩페이지
React
TypeScript
https://lillo.co.kr/

‘릴로’를 소개하는 랜딩 페이지 제작 프로젝트로, 전체 페이지를 React 기반으로 퍼블리싱하였습니다.
Scroll event, Button hover 등 다양한 인터랙션부터 메인 비주얼 영역의 애니메이션 작업까지 맡아, 브랜드 이미지를 효과적으로 전달할 수 있게 구현하였습니다.

사용자 인터랙션을 고려하여 hover 효과를 활용한 다양한 애니메이션을 구현하였습니다. 버튼, 카드 등의 요소에 애니메이션을 적용해 시각적인 몰입감과 사용자 반응성을 높였습니다.
